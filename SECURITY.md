# Security Policy

## Reporting Vulnerabilities
If you discover a security vulnerability, please email shopcolby@gmail.com.
Do not open a public issue.

## Supported Versions
We support the latest version deployed to production.

## Vulnerability Patching SLA
- Critical: Patched within 24 hours
- High: Patched within 7 days
- Medium: Patched within 30 days
- Low: Patched in next scheduled maintenance

## Dependencies
- Dependabot monitors for vulnerable dependencies weekly
- npm audit runs on every build
- Weekly automated security scans via GitHub Actions
